# Simple Interest Calculator

A simple web-based calculator for calculating simple interest and total accrued amount.

## Description

This HTML-based calculator allows users to calculate simple interest based on principal amount, annual interest rate, and time period. The calculator displays both the interest earned and the total amount accrued.

## Features

- Clean, responsive user interface
- Input validation for all fields
- Real-time calculation results
- Formula display for educational purposes
- Professional styling with hover effects

## Formula Used

**Simple Interest Formula:**
```
Simple Interest (I) = Principal (P) × Rate (r) × Time (t)
Total Amount (A) = Principal (P) + Interest (I)
```

Where:
- **A** = Total Accrued Amount (principal + interest)
- **P** = Principal Amount
- **r** = Annual Interest Rate (as a decimal)
- **t** = Time Period (in years)

## How to Use

1. Open the `simple-interest-calculator.html` file in any modern web browser
2. Enter the principal amount (initial investment)
3. Enter the annual interest rate as a percentage (e.g., 5 for 5%)
4. Enter the time period in years
5. Click the "Calculate Interest" button
6. View the results showing interest earned and total amount

## Installation/Usage

No installation required! Simply:
1. Download the `simple-interest-calculator.html` file
2. Open it in any web browser (Chrome, Firefox, Safari, Edge, etc.)
3. Start calculating!
